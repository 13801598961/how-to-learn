## 能力评测

### 能开发简单的移动APP
* 学习初级资料
* html(5)、css(3)、javasript基本技能
* 掌握html5移动开发流程
* 可以开发简单html页面
* 了解[盒子模型]
* [flex]布局

### 能开发常见APP
* 学习实战资料
* 进行前后台请求
* 能熟悉调用本地API
* 进行复杂页面布局
* 可以进行简单调试、错误定位
* 完成简单组件开发

### 开发出性能、交互较高的APP
* 学习高级资料
* 在社群基础上进行组件扩展
* 自行评估、引入第三方开源类库

## 如何参与贡献？

### 我有心得
我们认为分享是一种更好的学习方式。

学习过程中产生的心得以及认为比较好的资料，你可以按照下面流程进行提交：

* 点击右上角fork本文档
* 在自己的repositories中修改文档
* 检查修改内容是否正确
* 点击 Pull Request 提交修改到**whkfzyx**主干
* **whkfzyx**审核修改，接受Pull
